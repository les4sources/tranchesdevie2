# Prise en main

Bienvenue dans l'administration de **Tranches de Vie**. C'est ici que tu gères
tout : les commandes, les jours de cuisson, les mangeurs, les produits et les
comptes. Ce guide t'explique chaque écran, pas à pas, avec des captures.

## Se connecter

L'admin est protégé par un mot de passe unique, partagé entre les boulangers.

1. Va sur l'adresse de l'admin (elle finit par `/admin`).
2. Saisis le mot de passe.
3. Tu restes connecté·e pendant 24 h, puis il faut se reconnecter.

![L'écran de connexion](shot:login)

> Tu ne connais pas le mot de passe ? Demande-le à Michael — il n'est jamais
> écrit dans les SMS ni les mails.

## Le menu du haut

Tout se fait depuis la barre du haut. Voici à quoi sert chaque entrée :

| Entrée | Ce que tu y fais |
| --- | --- |
| **Commandes** | Voir, préparer et encaisser les commandes du jour. |
| **Jours de cuisson** | Les dates de fournée (mardi/vendredi) et la planification de la production. |
| **Mangeurs** | Les clients : fiches, portefeuille, historique, SMS. |
| **Produits** | Le catalogue : pains, variantes, prix, photos. |
| **Parties** | Les pizza parties publiques et les blocages de créneaux. |
| **Facturation** | Les factures et les paiements. |
| **€ (Revenus boulangers)** | La répartition des revenus entre boulangers et Les 4 Sources. |
| **📊 (Reporting)** | Les chiffres : refunds, payouts, parties. |
| **⚙️ (Paramètres)** | Farines, moules, capacités de production, notifications… |

![La barre de navigation de l'admin](shot:nav)

Sur téléphone, le menu se cache derrière le bouton **≡** en haut à droite.

## Par quoi commencer ?

- **Le matin d'une fournée** → onglet **Jours de cuisson** pour voir ce qu'il
  faut produire, puis **Commandes** pour préparer les sacs.
- **Un client au comptoir** → onglet **Mangeurs** pour retrouver sa fiche.
- **Changer un prix ou ajouter un pain** → onglet **Produits**.

Chaque chapitre de cette aide reprend un onglet en détail. Utilise le sommaire
à gauche pour naviguer.
