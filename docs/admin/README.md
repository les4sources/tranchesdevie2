# Bienvenue dans le guide de la boulangerie

Ce guide accompagne les boulangers de **Tranches de Vie** au quotidien dans
l'utilisation de leur espace d'administration en ligne.

Il est pensé pour être **lu à la volée**, entre deux fournées, sur un
téléphone ou sur l'ordinateur du fournil. Chaque page répond à **une seule
question pratique** : *« comment est-ce que je fais pour... »*.

## Comment utiliser ce guide

- **Vous ne connaissez pas encore l'espace d'administration ?** Commencez par
  [Premiers pas → Tour du propriétaire](premiers-pas/tour-du-proprietaire.md).
- **Vous cherchez à faire quelque chose de précis ?** Ouvrez le menu de
  gauche et repérez la section qui correspond à ce que vous voulez faire.
- **Un mot vous échappe ?** Le [glossaire](astuces/glossaire.md) reprend tous
  les termes que vous croiserez dans l'outil.
- **Un problème inattendu ?** Le mémo [Que faire si...](astuces/que-faire-si.md)
  liste les cas les plus fréquents et comment s'en sortir.

## Les sept espaces de votre administration

L'administration s'organise autour de sept grands espaces, accessibles depuis
le menu en haut de page :

| Espace | À quoi ça sert |
|---|---|
| **Commandes** | Voir toutes les commandes reçues, en enregistrer de nouvelles au comptoir, marquer les commandes prêtes ou récupérées, gérer les remboursements. |
| **Jours de cuisson** | Planifier vos fournées du mardi et du vendredi, vérifier que vous ne dépassez pas la capacité du four ou du pétrin. |
| **Mangeurs** | Consulter la fiche d'un client, ses commandes passées, l'état de son portefeuille, lui envoyer un message. |
| **Produits** | Ajouter un nouveau pain, modifier un prix, mettre un produit en pause. |
| **Facturation** | Éditer des factures, suivre ce qui a été facturé. |
| **Reporting** | Tableaux de bord des ventes, des remboursements, des revenus des artisans. |
| **Paramètres** | Réglages techniques : farines, moules, capacité du four, message envoyé quand une commande est prête. |

## Notre engagement de lisibilité

Cette documentation est mise à jour **automatiquement** à chaque évolution
de l'espace d'administration. Les copies d'écran sont regénérées, et le
texte est relu pour rester juste. Si vous trouvez une information périmée,
signalez-le à l'équipe : c'est un bug, pas une fatalité.
